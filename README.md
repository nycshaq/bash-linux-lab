# Title: Bash Linux Lab

## What it is:
A collection of Bash scripts built on an AWS EC2 Ubuntu instance to develop real Linux and systems administration skills.

## Scripts:
* syscheck.sh — system health snapshot: user, uptime, disk, memory
* process_check.sh — process monitor with timestamped logging, alerting, and cron scheduling
* backup.sh — automated directory backup with unique timestamps

## Environment: AWS EC2 t2.micro, Ubuntu, built and tested entirely via CLI
